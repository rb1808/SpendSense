# **App Name**: SpendSense

## Core Features:

- Dashboard & Spending Insights: Provide a clean dashboard displaying an overview of spending trends, budget progress, and key financial insights.
- Add & Categorize Expenses: Enable users to quickly input new expenses, including amount, date, and assign them to predefined or custom categories.
- Spending Visualizations: Display interactive charts and graphs to illustrate spending patterns and help users understand where their money goes over time.
- Monthly Budget Goal Setting: Allow users to set and track monthly budget goals for various spending categories, with clear progress indicators.
- Daily Spending Limit Alerts: Permit users to define a maximum daily spending limit and receive notifications when they are approaching or have exceeded it.
- Expense List Management: Offer a searchable and filterable list of all recorded expenses, allowing users to easily view, edit, or delete entries.
- AI-powered Transaction Categorization Tool: Automatically suggest and pre-fill expense categories for new transactions using an AI tool based on transaction descriptions and the user's historical spending habits.

## Style Guidelines:

- Primary color: Deep Teal (#1F8289) for a sense of clarity and professionalism, echoing the user's request for teal.
- Background color: Subtle Light Gray (#ECF4F5), derived from the primary's hue but heavily desaturated, providing a clean, airy base as requested.
- Accent color: Vibrant Green (#33BF72), an analogous hue to the primary teal, chosen for highlighting positive financial status, actions, and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and highly readable qualities, suitable for clear presentation of financial data.
- Utilize minimalist and clean line icons or filled modern icons that represent financial concepts, data, and tracking.
- Employ a modular, card-based layout on a clear grid system to organize information effectively and maintain a clean dashboard design.
- Incorporate subtle and smooth transitions for state changes, data updates in charts, and navigation to enhance user experience without distraction.
